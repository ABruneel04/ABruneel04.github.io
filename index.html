Project 1: Sale Price prediction using Linear Regression

Using Machine Learning, I explored the significance of a multitude of features in predicting the sale price of a given house. By the end of my work I was able to complete a Linear Regression model that performed as well as other top LR models (based on MSE). Worked with over 70 features, trimming down the amount of predictors with feature engineering and eventually using numerical values for correlation to evaluate features that would be used Took the list of correlated features and developed Linear Regression models to fit against the training data.

Project 3: Numerical Dynamo Modeling (Senior Thesis)

This was my first chance to implement Data Science principles into my work. I used Python to visualize parameters involved in dynamo generation for celestial bodies throughout our solar system. Using these visualizations, I was able to predict a given bodies’ potential for sustaining life. Used MagIC (an open-source code) to simulate the fluid physics occurring within planetary dynamos Compared the physics against benchmark cases that have been previously verified Varied the magnetic Prandtl parameter and determined if a dynamo could be sustained (and thus life) under those conditions Transferred data to python and created plots for each tested case Attached the thesis write-up explaining the physics and conclusions as well as the graphs in the GitHub repository linked here:
