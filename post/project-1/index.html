Project 1: Sale Price prediction using Linear Regression

September 15, 2022

Using Machine Learning, I explored the significance of a multitude of features in predicting the sale price of a given house. By the end of my work I was able to complete a Linear Regression model that performed as well as other top LR models (based on MSE).

  • Worked with over 70 features, trimming down the amount of predictors with feature engineering and eventually using numerical values for correlation to evaluate features that would be used
  • Took the list of correlated features and developed Linear Regression models to fit against the training data. Used p-values to determine if certain predictors would be included in the final model
  • Finally, developed interaction terms as well as quadratic terms and evaluated their significance. Insignificant terms were dropped using the hierarchical principle, and the final model was evaluated on MSE
